Differential Equations

A differential equation is an equation involving an unknown function \(y(t)\) and its derivatives \(y', y'', y'''\).
The order of a differential equation is the highest order derivative of \(y(t)\) in the equation.
\(y' + y = t^2\) is first order while \(y'' + ty' - y^2 = \cos(\pi t)\) is second order.
A differential equation is linear if it is of the form \(a_n(t)y^n + a_{(n-1)}(t)y^{(n-1)}+...+a_1(t)y'+a_0(t)y=f(t)\), where \(y^{n}=\frac{d^ny}{dt^n}\).
It is important to note that \(y, y', ..., y^{(n)}\) appear on their own, not inside any other function. Linear equations are important because they have nice properties, as we'll see later on.

A first order differential equation is separable if it is of the form \(y'=g(y)f(t)\)
Here are some examples of different differential equations with their respective charateristics.

\(y' = y^2\cos{(t)}\) First order, non-linear, separable DE

\(y'+yt=t\)    First order, linear, separable DE
\(y' = t-yt\)
\(y' = t(1-y)\)

\(y'+yt=t^2\)    First order, linear, non-separable DE
\(y'=t^2-yt\)
\(y'=t(t-y)\)

A first order equation is autonomous if it is of the form \(y'=g(y)\) (i.e. there is no t explicitly in the equation)
\(y'=y^2+1\)    First order, non-linear, autonomous

Before we start developing techniques to solve differential equations, note that is is usually impossible to solve differential equations in real life;
by solve I mean find an explicit formula for \(y(t)\) but we can always use computers to approximate.


Separable Equations

\(y' = g(y)f(t)\)
=> \(\frac{1}{g(y)}y' = f(t)\)
\(\int\frac{1}{g(y)}y'dt = \int f(t)dt\)    Substitute \(y'=\frac{dy}{dt}\)
\(\int\frac{1}{g(y)}dy = \int f(t)dt\)    Solve for \(y\)

Example: \(\y'=y^2t => \int\frac{1}{y^2}dy = \int tdt)
-\frac{1}{y} = \frac{t^2}{2}+C
y(t) = \frac{-1}{\frac{t^2}{2}+C}     C\epsilon\Bbb{R}\)
